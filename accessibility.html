Our commitment to accessibility

We aim to develope a website which enables anyone to use all of the functions

We address our accessibility in following ways

  • Accesskey for every page, every function, every link and every figure in this website. Everyone can use all of functions in this website without a mouse
  • If the picture is not clear enough or just too small, users can see the full size of the figure by clicking it or using accesskey
  • If the picture is not able to load, the user will see a text description of the picture instead
